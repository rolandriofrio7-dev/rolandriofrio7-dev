## 🩺 What I'm building

**[IO — Caregiver Coordination](https://github.com/rolandriofrio7-dev/care-command-center)** — my flagship project
A non-diagnostic caregiver coordination prototype built from real clinical experience. Tracks whether care-plan tasks happened on time, surfaces drift, and keeps a human in charge of every judgment. Python/FastAPI backend with a deterministic state machine and full provenance, Next.js/React/TypeScript frontend, automated test coverage. Synthetic-only, non-diagnostic by design.

**Vital-signs CSV visualizer** *(in development)*
Reads vital-sign data from CSV and displays it intuitively, using contextual sliders to frame readings for review.

**Creative tooling for editors** *(in development)*
Custom scripting tools that speed up repetitive work for video editors.
